Sobre o Stylus

Stylus é um pré-processador CSS poderoso porém, simples. A idéia é que com ele possa ser criado CSS de forma ágil, legível e simples.

Exemplo:

border-radius()
  -webkit-border-radius arguments
  -moz-border-radius arguments
  border-radius arguments

body
  font 12px Helvetica, Arial, sans-serif

a.button
  border-radius(5px)

Código CSS resultante:

body {
  font: 12px Helvetica, Arial, sans-serif;
}
a.button {
  -webkit-border-radius: 5px;
  -moz-border-radius: 5px;
  border-radius: 5px;
}

Instalação

Stylus, tal como Less, utiliza o NodeJS como dependência e sua instalação é extremamente simples

$ npm install stylus -g

Utilização

Para renderizar o css 

stylus style.styl

Para observar um arquivo por modificações em tempo real 

stylus -w style.styl

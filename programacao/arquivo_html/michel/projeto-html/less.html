Começando...

Less é um pré-processador CSS, isso significa que estende a linguagem CSS, adicionando funcionalidades que permitem variáveis, mixins, funções e muitas outras técnicas que permitem que você faça CSS que é de manutenção mais fácil, tematizáveis e extensível.

Por exemplo:

@base: #f938ab;

.box-shadow(@style, @c) when (iscolor(@c)) {
  -webkit-box-shadow: @style @c;
  box-shadow:         @style @c;
}
.box-shadow(@style, @alpha: 50%) when (isnumber(@alpha)) {
  .box-shadow(@style, rgba(0, 0, 0, @alpha));
}
.box {
  color: saturate(@base, 5%);
  border-color: lighten(@base, 30%);
  div { .box-shadow(0 0 5px, 30%) }
}

Após processado, será:

.box {
  color: #fe33ac;
  border-color: #fdcdea;
}
.box div {
  -webkit-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
}

Instalação

O modo mais simples para instalar o Less no servidor, é via npm, um gerenciador de pacotes do NodeJS, veja abaixo como instalá-lo:

$ npm install -g less

Usando em linha de comando

Uma vez instalado, você pode chamar o compilador via terminal com o seguinte comando:

$ lessc styles.less

Com isso, irá exibir o código CSS no terminal. Para salvar a saida CSS para um arquivo, segue o exemplo abaixo:

$ lessc styles.less styles.css

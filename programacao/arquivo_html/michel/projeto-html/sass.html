Sobre o SASS

Sass é o mais maduro, estável e poderoso pré-processador CSS do mercado hoje

Compatibilidade

Sass é completamente compatível com todas as versões do CSS. Também pode ser utilizado qualquer biblioteca CSS disponível.

Rico

Sass possuí mais características e habilidades do que qualquer outro pré-processador CSS. Sua equipe de desenvolvedores não somente o mantém atualizado como também trabalha para sempre adicionar novas ferramentas

Maduro

Sass já possui 9 anos de mercado


Aprovado pela indústria

Diversas empresas utilizam em seus projetos, como Twitter, em seu projeto Bootstrap

Comunidade

Sass é ativamente mantido e desenvolvido por um consórcio de diversas empresas de tecnologia e centenas de desenvolvedores

Frameworks

Existem incontáveis frameworks desenvolvidos com Sass em seu código. Compass, Bourbon, e Susy só para listar alguns.


Instalação

Linux

Se você está utilizando uma distribuição Linux, precisa instalar Ruby primeiro. Pode instalar o Ruby através do gerenciador de pacotes de sua distribuição

sudo su -c "gem install sass"

Windows

Antes de utilizar o Sass, você precisa instalar o Ruby. O modo mais rápido de obter Ruby para seu computador com Windows é utilizando o instalador. É uma aplicação de 1 clique que irá prover tudo o que precisa.

O instalador também instalará Ruby para linha de comando (via PowerShell) que irá lhe permitir instalar as bibliotecas necessárias.

Mac

Se preferir a linha de comando em vez de aplicações GUI, é um processo muito simples. Sass depende do Ruby, e Ruby já está pré-instalado no Mac

Instalando Sass

Este é o meio mais rápido para iniciar o uso do Sass via linha de comando:

  1. Abra o terminal
  2. Instale o Sass: Ruby Gems possui um gerenciador de pacotes para instalar diversas ferramentas, como o Sass. Digite: 
    gem install sass
    Este comando instalará Sass e suas dependências. É como mágica. Se receber alguma mensaem de erro, use o comando "sudo" 
    sudo gem install sass
  3. Checar instalação 
    sass -v
